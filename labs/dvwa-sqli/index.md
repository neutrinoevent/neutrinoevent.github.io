# DVWA SQLi Lab

This lab is part of my cybersecurity portfolio and focuses on identifying and exploiting SQL injection vulnerabilities in the Damn Vulnerable Web Application (DVWA).

---

## 🚀 How to Run Locally

You can spin up DVWA using Docker:

```bash
docker run --rm -it -p 8080:80 vulnerables/web-dvwa
```

Then open your browser and go to:

[http://localhost:8080](http://localhost:8080)

### Default Login Credentials:
- **Username:** `admin`
- **Password:** `password`

---

## 🔍 Objectives

- Understand basic SQL injection techniques
- Bypass login forms
- Dump database tables via SQLi
- Learn about security levels in DVWA
