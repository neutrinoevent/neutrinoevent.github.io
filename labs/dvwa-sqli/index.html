DVWA - SQL Injection (Low Security)

Goal: Demonstrate basic SQL injection and suggest mitigation.

🧩 Setup

  • Docker image: vulnerables/web-dvwa
  • Security level: Low

🚨 Exploit

admin' OR '1'='1' --

✅ Result

Bypasses login, authenticates as admin.

🛡️ Mitigation

  • Parameterized queries (e.g., mysqli_prepare())
  • Input sanitization
  • WAF (e.g., mod_security)
